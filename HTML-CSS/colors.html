We can choose between 3 ways to use colors on our code:

  • Type the name of a color (a valid one)
  • Use hexadesimal sintaxis to pick a color
  • Use rgb[a] sintaxis to pick a color. This optional [a] can help with the transparency, putting an extra "," then choosing a value between 0 and 1
